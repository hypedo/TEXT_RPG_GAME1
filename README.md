# Java 콘솔 RPG 게임

## 프로젝트 소개

이 프로젝트는 Java를 사용하여 제작한 간단한 콘솔 기반 RPG 게임입니다.
사용자는 영웅의 이름을 입력한 뒤 게임에 입장하며, 사냥터에서 몬스터와 전투하거나 포션 상점에서 능력치를 강화할 수 있습니다.

기본적인 RPG 게임 구조인 **영웅 상태 관리, 몬스터 전투, 경험치 획득, 레벨업, 골드 보상, 포션 구매 시스템**을 구현하는 것을 목표로 하였습니다.

## 개발 환경

* Language: Java
* IDE: Eclipse / IntelliJ IDEA / VS Code 등 Java 실행 가능 환경
* 실행 방식: 콘솔 입력 기반 프로그램

## 주요 기능

### 1. 영웅 생성

게임 시작 시 사용자가 영웅의 이름을 입력합니다.
영웅은 기본 능력치를 가진 상태로 게임을 시작합니다.

초기 능력치는 다음과 같습니다.

| 항목  | 초기값 |
| --- | --: |
| 레벨  |   1 |
| 힘   |  15 |
| 체력  |  80 |
| 방어력 |  25 |
| MP  |   0 |
| 경험치 |   0 |
| 골드  |   0 |

## 2. 상태 출력

게임 진행 중 현재 영웅의 상태를 확인할 수 있습니다.

출력되는 정보는 다음과 같습니다.

* 영웅 이름
* 레벨
* 힘
* 방어력
* HP
* MP
* 경험치
* 보유 골드

## 3. 장소 선택

사용자는 게임 내에서 다음 장소를 선택할 수 있습니다.

| 번호 | 장소    |
| -: | ----- |
|  1 | 사냥터   |
|  2 | 포션 상점 |
|  0 | 게임 종료 |

## 4. 사냥터 시스템

사냥터에 입장하면 전투할 몬스터를 선택할 수 있습니다.

현재 구현된 몬스터는 다음과 같습니다.

| 몬스터 | 레벨 | 공격력 |   HP | 방어력 | 경험치 보상 | 골드 보상 |
| --- | -: | --: | ---: | --: | -----: | ----: |
| 너구리 |  1 |  20 |  100 |   5 |     10 |    10 |
| 살쾡이 |  5 | 100 | 2000 |  20 |     50 |    30 |

## 5. 전투 시스템

전투 중 사용자는 두 가지 행동을 선택할 수 있습니다.

| 번호 | 행동    |
| -: | ----- |
|  1 | 일반 공격 |
|  2 | 파괴광선  |

### 일반 공격

영웅의 일반 공격 데미지는 다음 공식으로 계산됩니다.

```java
hero_level * 10 + hero_power * 30
```

몬스터는 자신의 방어력만큼 데미지를 감소시킨 뒤 HP가 감소합니다.

### 파괴광선

파괴광선은 MP가 20 이상일 때 사용할 수 있는 스킬입니다.

스킬 데미지는 다음 공식으로 계산됩니다.

```java
hero_power * 100 + hero_level * 30
```

스킬 사용 후 MP가 감소합니다.
MP가 부족한 경우 스킬을 사용할 수 없습니다.

## 6. 몬스터 공격

영웅이 공격한 뒤 몬스터가 살아있으면 몬스터가 반격합니다.

몬스터의 공격력과 영웅의 방어력을 비교하여 영웅의 HP가 감소합니다.

```java
hero_hp = hero_hp + hero_defense - monster_power;
```

영웅의 방어력이 몬스터의 공격력보다 높거나 같으면 데미지를 받지 않습니다.

## 7. 사망 및 부활 시스템

전투 중 영웅의 HP가 0 이하가 되면 사망 메시지가 출력됩니다.
이후 영웅은 기본 HP인 80으로 부활합니다.

```java
hero_hp = 80;
```

## 8. 보상 시스템

몬스터를 처치하면 해당 몬스터의 경험치와 골드를 획득합니다.

예시:

```java
hero_exp += monster_exp;
hero_gold += monster_gold;
```

## 9. 레벨업 시스템

영웅의 경험치가 현재 레벨 기준 필요 경험치 이상이 되면 레벨업합니다.

레벨업 조건은 다음과 같습니다.

```java
hero_exp >= hero_level * 80
```

레벨업 시 다음 효과가 적용됩니다.

* 레벨 1 증가
* 경험치 0으로 초기화
* 레벨업 보상 골드 지급

골드 보상 공식은 다음과 같습니다.

```java
hero_gold += hero_level * 50;
```

## 10. 포션 상점

포션 상점에서는 골드를 사용하여 영웅의 능력치를 강화할 수 있습니다.

| 번호 | 아이템       |   가격 | 효과      |
| -: | --------- | ---: | ------- |
|  1 | 힘 증강 포션   |  30원 | 힘 +3    |
|  2 | 방어력 증강 포션 |  30원 | 방어력 +3  |
|  3 | 경험치 증강 포션 | 100원 | 경험치 +50 |
|  4 | HP 증강 포션  |  10원 | HP +50  |
|  5 | MP 증강 포션  |  10원 | MP +50  |

골드가 부족한 경우 구매할 수 없으며, “구매불가” 메시지가 출력됩니다.

## 코드 구조

주요 메서드는 다음과 같습니다.

| 메서드명                 | 설명                                |
| -------------------- | --------------------------------- |
| `main()`             | 게임 실행, 사용자 입력, 장소 선택, 전체 게임 흐름 관리 |
| `hero_attack()`      | 영웅의 일반 공격 데미지 계산                  |
| `hero_skill()`       | 영웅의 스킬 공격 처리                      |
| `monster_attack()`   | 몬스터의 공격 처리                        |
| `hero_attacked()`    | 영웅이 몬스터에게 공격받았을 때 HP 감소 처리        |
| `monster_attacked()` | 몬스터가 영웅에게 공격받았을 때 HP 감소 처리        |
| `potionStore_show()` | 포션 상점 구매 기능 처리                    |
